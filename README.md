## Overview

This project is an interactive Management Information System (MIS) dashboard built in **Microsoft Excel** to monitor and analyze sales operations performance.
The dashboard provides management-level insights using dynamic reporting, KPI tracking, and filter-based analysis.

It simulates a real corporate reporting file used by MIS Executives and Operations/Reporting Analysts.

---

## Key Features

### KPI Monitoring

The dashboard displays important business metrics:

* **Total Revenue**
* **Average Achievement %**
* **Total Orders / Transactions**

All KPI values update automatically based on slicer selections.

---

### Performance Analysis

The report allows managers to evaluate business performance through:

* Employee performance ranking
* Identification of underperformers (Bottom 5 employees)
* Region-wise sales performance comparison
* Product category performance tracking

---

### Interactive Filtering

Users can dynamically filter reports using slicers:

* Month
* Region
* Product Category

Selecting any filter updates:

* KPIs
* Charts
* Rankings
* Reports

---

## Dashboard Components

### 1. KPI Section

Top-level summary showing overall business health.

### 2. Employee Ranking Report

Ranks employees based on performance and revenue contribution.

### 3. Bottom 5 Report

Highlights lowest-performing employees for management attention.

### 4. Region Report

Compares sales performance across regions (Central, East, South, West).

### 5. Category Performance

Analyzes revenue and achievement across product categories:

* Furniture
* Office Supplies
* Technology
